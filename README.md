# 原神活动日历

## 版本

v0.0.1<br>
 ⚠ 适配nonebot2-2.0.0beta2版本；

## 安装

通过 `nb`或 `pip` 命令安装插件

`pip install nonebot_plugin_covid`
<br>或<br>
`nb plugin install nonebot_plugin_covid`

## 命令

`原神日历` 查看本群订阅服务器日历<br>
`原神日历 on/off` 订阅/取消指定服务器的日历推送<br>
`原神日历 time 时:分`  设置原神日历推送时间<br>
`原神日历 status`  查看原神日历推送设置
